# jmsg-server

A text based Java message server using Java Networking.

![Application Image](jmsg-server.png)

## Building

Prerequisites
- Java JDK
- Gradle Build Tool

```bash
git clone https://github.com/realeigenvalue/jmsg-server.git
cd jmsg-server
gradle build
gradle run
```
